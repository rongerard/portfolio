Project Description

This web app is built using the MERN stack, but with MySQL replacing MongoDB. Collaboration was crucial, as the project was large-scale and required numerous revisions, ultimately taking three months to complete. Through this experience, I learned the importance of flexibility and teamwork. My key contributions included implementing the SMS API using Semaphore to send automated appointment reminders, developing a QR scanner that updates the database, and creating the report tab. Additionally, I handled Gmail OTP verification and the forgot password function while assisting with various other features. I also utilized Selenium WebDriver for automated testing to ensure the reliability and efficiency of the application.
