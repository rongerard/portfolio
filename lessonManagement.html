Project Description

This website is an online learning platform for C#, featuring an admin panel for managing lessons, quizzes, assessments, and users. The project utilizes XAMPP as the local server, with JavaScript and PHP as the primary programming languages. It also incorporates the jQuery library and XML for data storage, using AJAX as a bridge for seamless communication.
